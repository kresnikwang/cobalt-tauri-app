Engineered for Desktop Excellence

A modern toolkit designed specifically to overcome Webview limitations, with premium Native optimizations built right in.

macOS ARM64 Stability

Specifically tuned for modern macOS targets. Spawning a compiled Rust core bypasses Electron's V8 CodeRange virtual memory crashes (prevalent on ARM64 macOS machines with 64KB pages).

Native Rust Downloader

Tauri Rust processes manage the actual multi-threaded scheduler and download pipelines natively. Progress updates and download speeds are sent cleanly via zero-lag IPC events.

Clipboard Link Monitor

Includes a background thread loop that monitors the system clipboard. When a video/audio link is copied, a premium toast slides up inside the application so you can download in one click.

FFmpeg Remux Tool

Includes a dedicated Remuxing utility view. Drag and drop local audio or video files to repackage them instantly into MP4, WebM, MKV, MP3, etc. with zero re-encoding times.

Smart Fallbacks & Proxies

Configured with InnerTube fallback mechanisms for smooth YouTube downloads, and complete support for HTTP proxies (e.g. Clash, Clash Verge) to access restricted content.

Multilingual Interface

Fully translated into English, Chinese (中文), and Russian (Русский). The app dynamically changes language terms instantly using Svelte 5 dynamic translation store.

Decoupled & Clean Architecture

Electron apps run backend node logic inside browser contexts. Cobalt Desktop takes a native, sandboxed approach by segregating responsibilities:

  • Svelte 5 Runes Viewport: A minimal, highly performant Webview layer rendering interactive reactive state without accessing system storage.
  • Tauri Rust Core Scheduler: Coordinates downloading buffers, settings write/read, folder picker window dialogs, and subprocess health monitoring.
  • Local Cobalt API Subprocess: Rust launches an independent Node.js Cobalt API instance on port 47301, automatically killing it on exit.
